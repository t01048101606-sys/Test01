# 마크다운(Markdown)이란?

마크다운은 **텍스트 기반의 마크업 언어**입니다. 복잡한 HTML 태그 없이 몇 가지 특수문자만으로 글의 구조와 스타일을 멋지게 꾸밀 수 있습니다.

> **💡 마크다운의 최대 장점**
> 1. 문법이 매우 쉽고 직관적입니다.
> 2. 마우스 없이 키보드만으로 서식을 지정할 수 있어 생산성이 높습니다.
> 3. 벨로그(Velog), 노션(Notion), 깃허브(GitHub) 등에서 필수적으로 사용됩니다.

---

## 자주 쓰는 핵심 마크다운 문법

### 1. 텍스트 강조 (Emphasis)
* **굵게 만들기**: 단어 앞뒤로 별표 2개 `**`를 붙입니다. (예: **중요한 내용**)
* *기울이기*: 단어 앞뒤로 별표 1개 `*`를 붙입니다. (예: *이탈릭체*)
* ~~취소선~~: 단어 앞뒤로 물결 표시 2개 `~~`를 붙입니다. (예: ~~지워진 텍스트~~)

### 2. 제목 (Headers)
해시 기호(`#`)의 개수로 글의 제목과 계층 구조를 정합니다. `#`이 적을수록 큰 제목입니다.
# 1단계 제목 (가장 큼)
## 2단계 제목
### 3단계 제목

### 3. 목록 (Lists)
* **순서 없는 목록**: `-`, `*`, `+` 중 하나를 쓰고 한 칸 띄웁니다.
    * 이렇게 들여쓰기를 하면 하위 목록이 됩니다.
* **순서 있는 목록**: 숫자와 마침표(`1.`)를 쓰고 한 칸 띄웁니다.
    1. 첫 번째 할 일
    2. 두 번째 할 일

### 4. 링크와 이미지 (Links & Images)
* **링크**: `[텍스트](URL 주소)` 형식으로 작성합니다. (예: [구글 사이트](https://google.com))
* **이미지**: 링크 형식 앞에 느낌표(`!`)만 붙여주면 됩니다. `![이미지 설명](이미지 링크)`

### 5. 코드 블록 (Code Blocks)
* **인라인 코드**: 문장 중에 `이렇게` 넣을 때는 앞뒤로 백틱(`` ` ``)을 한 개씩 씁니다.
